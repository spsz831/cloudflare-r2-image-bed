# 团队图床使用指南

## 📸 图床地址
**https://00307e36.image-bed-frontend.pages.dev**

## 🚀 如何使用

### 1. 上传图片
- **方式1**：直接将图片拖拽到上传区域
- **方式2**：点击"点击选择文件"按钮选择图片
- **方式3**：可以一次选择多个图片批量上传

### 2. 支持的格式
- JPG/JPEG
- PNG
- GIF
- WebP
- SVG

### 3. 文件限制
- 单个文件最大 50MB
- 总存储空间 10GB（免费额度）

### 4. 获取链接
上传完成后，可以复制三种格式的链接：
- **直链**：直接访问图片的URL
- **Markdown**：适用于 Markdown 文档
- **HTML**：适用于网页代码

### 5. 查看历史
- 在页面下方可以查看上传历史
- 历史记录保存在本地浏览器中
- 可以快速复制之前上传图片的链接

## ⚠️ 注意事项
- 这是公开图床，任何人都可以上传
- 上传的图片会被公开访问
- 请不要上传敏感或私人信息
- 建议团队内部协调使用，避免滥用
